# 安心猫首页重设计 · 桌宠主场(A) 设计文档

日期:2026-06-15
范围:仅首页(`/`)。分诊 / 报告 / 问答 / 知识 / 毛孩子页本期不动。

## 背景与问题

刚上线收到用户反馈三条:① UI 太丑 ② 交互路径不清晰、点进去不知道咋操作 ③ 不清楚有哪些功能。

根因:首页桌宠化把功能入口(看病 / 聊天 / 小知识)藏成了「猫坐下才冒的静态心事泡」,界面不自解释 —— 新人进来只看到一只猫在屋里溜达,不知道这 App 干嘛、点哪。这与之前「守门橘猫」产品调研第〇批早已点名的"入口发现性短板"一致。

注:用户朋友给的参考图是宠物**电商 / 社区** App(商品卡、社区帖、橙色堆满功能格),属不同品类。本设计**不照搬**,守住「猫咪安心分诊器 —— 医疗、克制、暖奶白、降焦虑」的定位。

## 目标

1. 首页一眼自解释:3 秒懂"这是猫咪分诊器"、立刻知道点哪(治反馈②③)。
2. 保留桌宠的情感差异化(产品独特价值,别的分诊器没有)。
3. 视觉精致度提升:克制暖奶白 + 陶土红,不堆砌(治反馈①)。

## 非目标 / 红线

- 不做电商 / 导流(含医院推荐)/ 社区 / 养成(经验值 / 解锁 / 商城);不照搬电商宠物 UI。
- 风险三色(红 / 黄 / 绿)独立信号层不动,不被背景 / 装饰暖色污染。
- 桌宠搭话不制造焦虑、不出诊断口吻("绝不你猫是不是病了")。
- 本期只重做首页;不碰分诊判级 / 报告 / 医学文案。

## 设计:A 桌宠主场

首页自上而下 4 块,暖奶白克制底:

### ① 问候条(精简)
头像 + 「早上好,哈基米 · 6 个月 · 母」一条。原大资料卡的 疫苗 / 绝育 / 相册 / 生活照 移回「毛孩子」(`/onboarding` 档案)页管理。首页上半立刻清爽,把空间让给桌宠和功能。

### ② 价值主张 + 看病主入口
一行价值主张「猫不对劲?选症状,30 秒红黄绿就医建议」+ 一个陶土红(`--accent`)「看病」大按钮 → `/symptoms`。这是首页第一眼的功能锚,3 秒懂干嘛、知道点哪。

### ③ 桌宠院子(主角)
两层入口分工:**顶部明牌 = 理性保底(随时找得到);猫搭话泡 = 感性生动(它来邀请你)**。

- **顶部常驻明牌入口 pills**:看病(accent)/ 聊天 / 小知识,毛玻璃质感,浮在院子上方常驻可达。链 `/symptoms`、`/behavior`、`/knowledge`。不依赖猫的任何状态。
- **去掉静态心事泡**(原 `thinking` 三泡渲染移除)。
- **新增「小猫搭话泡」**:猫漫游 / 坐着时,随机间隔冒一个带功能意图的对话泡;点泡跳对应功能,不点 N 秒自动收。话术温和、守降焦虑红线。话术池(每功能 2–3 句,随机轮):
  - → `/behavior`(聊天):「你是来找我聊天的吗?」「有啥想问的,我都在哦~」
  - → `/knowledge`(小知识):「教你认几个『看着吓人其实不用慌』的?」
  - → `/symptoms`(看病,温和):「它今天精神还好吗?有不放心点我看看」
- 搭话泡与现有「摸猫 / 互动猫语」泡**区分**:搭话泡可点跳转(右下角箭头暗示可点),互动猫语泡只是反馈、不跳转。
- 院子内**全部互动保留**:漫游、家具(窝 / 抓板 / 纸箱 / 毛线球 / 水碗 / 地毯)、道具栏(梳子 / 水瓶 / 逗猫棒)、摸 / 梳毛 / 喂水 / 逗 / 洗脸 / 钻箱 / 晒太阳。

### ④ 最近记录
保留,每条三色档位圆点(独立信号层,`var(--red/amber/green)`,不动)。

### 视觉精致(治"丑")
统一圆角(`--radius-card`)与阴影层次、克制留白、标题衬线 + 功能黑体的字阶、入口 pills 毛玻璃精致化。整体守暖奶白(`--paper/--surface`)+ 陶土红(`--accent`)点缀,绝不堆成电商风。不新增设计令牌。

## 实现要点(涉及文件)

- `src/app/page.tsx`:首页主体 —— 问候条精简、价值主张 + 看病主入口、院子顶部明牌 pills、最近记录;院子内去 `thinking` 心事泡、加「搭话泡(invite)」机制(随机调度 + 话术池 + 点击跳转 router.push + 与 `talk` 反馈泡区分)。
- 资料卡内容(疫苗 / 绝育 / 相册 / 生活照)迁移到「毛孩子」/ `/onboarding` 页;首页只读一条精简档案。
- 视觉 token 复用 `src/app/globals.css`,不新增令牌。
- 不动:分诊 / 报告 / 问答 / 知识页、判级引擎、医学文案、三色令牌、桌宠互动逻辑与素材。

## 验收标准

- 慌乱测试:首页 4 秒内能找到「看病」主入口 + 一句价值主张说清干嘛。
- 顶部明牌 pills 常驻可达,不依赖猫坐下。
- 搭话泡:随机冒、点了正确跳转(聊天 / 知识 / 看病)、话术不制造焦虑;不点自动收;与互动猫语泡不混。
- 三色圆点观感不受背景 / 装饰影响。
- 360 / 393 / 430 三档宽度适配仍正常(不裁不留白)。
- 桌宠全部互动仍工作(摸 / 梳 / 喂 / 逗 / 洗脸 / 钻箱 / 漫游)。
- `npm run build` 通过。
