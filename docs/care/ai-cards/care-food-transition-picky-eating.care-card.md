```yaml
    - 完全不吃或明显食欲下降
    - 呕吐、腹泻、血便、黑便

    - 体重下降或精神差
```

## 建议口径

- 先确认是真的挑食,还是不舒服导致不想吃。
- 换粮至少 5-7 天过渡:旧粮多、新粮少,每天慢慢增加新粮比例。
- 曾经软便的猫可以把过渡拉到 10-14 天。
- 不要今天 A、明天 B、后天 C 地频繁试,会让猫更挑,肠胃也更乱。
- 定时放饭、到点收走,比全天不断添新口味更容易稳定习惯。
- 可以用少量温水、原本爱吃的湿粮香气辅助,但不要把零食拌成主食。

## 不建议

- 不要为了让它吃,长期只喂零食或冻干。
- 不要突然整袋换新粮。
- 不要把“完全不吃”当挑食拖着观察。
