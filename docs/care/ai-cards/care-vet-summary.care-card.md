# 日常养育知识卡:整理给医生看的病情说明

```yaml
care_id: care_vet_summary
entry_topics:
  - 整理给医生看
  - 整理给兽医看
  - 帮我整理给医生
  - 帮我整理给兽医
  - 帮我把情况说清楚
  - 写给医生
  - 写给兽医
  - 病情说明
  - 给医生看的话
  - 给医生发
  - 给兽医发
  - 带去医院
  - 整理就医说明
  - 整理症状描述
  - 给医生说
  - 给兽医说
  - 一会儿要去医院
  - 准备就医
ai_task:
  primary_goal: |
    按下面 output_rules / content_order / hard_rules 整理一段主人能直接复制
    给医生 / 兽医的中文说明。目的是帮主人把刚才聊的情况说清楚,不是 AI 替
    医生下诊断。
  tone: 主人对医生说话的第一人称语气,不是医生病历格式,也不是 AI 客服。
output_rules:
  format: 自然语言段落 2-4 段;不要用【主诉】【现病史】【既往史】这种病历字段标签。
  length: 120-260 字。
  opening: 「医生您好,我家猫……」开头。
  closing: 不要在末尾再写「仅供参考 / 不能替代兽医」 — 页面底部已有常驻提示。
content_order:
  - 主要症状(简短一句:核心症状 + 持续时间)
  - 变化趋势(从哪天开始 / 加重 / 缓解 / 没变)
  - 精神 / 吃喝(精神状态、食欲、饮水量)
  - 排尿排便(频次 / 性状 / 量;不清楚就「暂未说明」)
  - 伴随表现(其他征兆;没有就明确写「没有 X / Y」)
  - 已采取处理(用过的药、护理动作;没有就「暂未给药 / 暂未处理」)
  - 猫咪基础信息(月龄 / 性别 / 绝育状态 / 疫苗 / 驱虫;从猫资料拿,缺就「暂未说明」)
  - 想请医生重点看什么 / 想确认什么
hard_rules:
  - 不诊断:不写「确诊 X 病」「高度怀疑 X」「就是 X 感染」「考虑 X」。
  - 不写病原英文缩写或具体病名:禁词包括 FCV / FHV / FVR / FPV / calicivirus /
    herpesvirus / Bordetella / Chlamydia / Mycoplasma / Cryptococcus /
    Pasteurella / 杯状病毒 / 疱疹病毒 / 支原体 / 衣原体 / 博德特氏菌 /
    隐球菌 / 巴氏杆菌。只能说大类「上呼吸道感染 / URI」「胃肠炎」「中毒」等。
  - 缺资料用「暂未说明」「暂不清楚」,不要编造日期 / 数字 / 既往史。
  - 不写药品名 / 剂量 / 商品名 / 购买建议;最多说「想请医生评估是否需要用药」。
  - 如果上游 current_tier=red,语气保持急迫:「我们正在准备前往医院 / 急诊」
    开头。禁用「不是急诊 / 黄色预警 / 黄色信号 / yellow flag / 别等但不急 /
    今天去就行 / 先观察」等降级措辞。
  - 如果 current_tier=yellow,用「这几天内挂号 / 尽快就医 / 不要拖」。
  - 不在文末写免责声明。
questioning_boundary:
  ask_first: []
  enough_to_advise_when:
    - 用户明确说要整理给医生 / 兽医看的描述,直接基于现有 chat 上下文 + 猫
      资料整理,不再多轮追问。
health_switch:
  switch_to_medical_when:
    - 用户在整理过程中又补充新症状,且涉及红旗信号(张口喘 / 大量出血 /
      抽搐 / 尿不出 / 误食高危物 / 持续 24h 不吃)。
```

## 合规输出参考样例(给 LLM 抄格式不抄字)

```
医生您好,我家猫小橘 5 天前(5/23)开始打喷嚏,起初每天 5–6 次,
今天早上鼻涕变成黄绿色,精神也明显萎靡,躲在床底叫不出来。

吃喝方面:今早拒食,加热罐头只吃了几口,平时一次能吃完;饮水比平时少。
排尿排便暂未说明。

伴随表现:没有张口喘、口腔溃疡、抽搐或牙龈发紫。
已采取:5/26 用生理盐水棉签擦过鼻部分泌物,5/27 早加热罐头(食欲未恢复),
未自行给药。

猫咪情况:20 月龄,公猫,已绝育;5/18 完成本年妙三多 + 狂犬;5/1 博来恩 +
大宠爱内外驱虫。家里 5/22 接回 3 月龄幼猫,未隔离,共用食盆和猫砂。

想请医生重点看:是否需要做 PCR 或拭子确认病原方向、当前最该做的检查,以及
是否需要开抗生素配合治疗。
```

## 关键禁忌(再次强调)

- 禁直接报具体病原名(无论中文 / 英文 / 缩写)。
- 禁药品名、剂量、推荐购买链接。
- 红档场景禁任何降级措辞,语气保持「立刻就医 / 急诊」级别。
- 禁文末再写「不能替代兽医」类免责声明 — 页面底部已有常驻 Disclaimer。
