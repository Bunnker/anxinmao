# 日常养育资料库

这个目录放非医疗场景的本地资料卡,用于 `/api/behavior` 的日常行为、喂养、训练和环境建议。

定位:

- 给 AI 稳定的日常养猫口径,减少随机发挥。
- 和 `docs/medical` 分开,避免普通问题被医疗化。
- 只提供行为/喂养/护理方法;一旦出现疼痛、食欲精神异常、排尿异常、呼吸异常等健康信号,切回医学分诊资料。
