# 一、呕吐 / 腹泻 / 不吃东西

## 1.1 呕吐

### 🔴 红旗(立刻 / 尽快送医)
- 24 小时内呕吐超过 2 次。〔VCA —— "如果宠物在 24 小时内呕吐超过两次……应前往紧急护理"。https://vcahospitals.com/urgent-care/health-concerns/diarrhea-or-vomiting〕
- 呕吐物中带血。〔Merck —— 呕吐若伴呕血、腹痛、精神沉郁、脱水、虚弱、发热、体重下降,需兽医检查。https://www.merckvetmanual.com/cat-owners/digestive-disorders-of-cats/vomiting-in-cats〕
- 呕吐同时伴精神萎靡 / 虚弱、食欲明显下降、饮水或排尿异常,或同时腹泻。〔Cornell —— 出现嗜睡、虚弱、食欲下降、呕吐物带血、口渴增加、排尿增多或减少、或同时腹泻的猫应及时由兽医评估。https://www.vet.cornell.edu/.../vomiting〕
- 反复或持续呕吐(尤其疑似误食异物时)—— 可能胃肠道梗阻。〔VCA —— 反复或持续呕吐 / 腹泻可能是中毒、肠道梗阻或急性胃肠道感染征兆。https://vcahospitals.com/know-your-pet/emergencies-in-cats〕(注:v0.1 原作「干呕、想吐吐不出」,所引页只说"呕吐",已收回。)
- 怀疑误食线 / 丝线 / 彩带类长条物(线性异物):呕吐、不吃、脱水、精神差为典型表现,急症。〔VCA —— 线性异物最严重并发症是肠穿孔致腹膜炎、可危及生命;常见表现含呕吐、厌食、脱水、嗜睡。https://vcahospitals.com/know-your-pet/linear-foreign-body-in-cats〕
- 看到口腔或肛门处露出线头,**绝对不要拉**,直接送医。〔⚠️ **来源对不上** —— v0.1 原引 Merck 急救页,核查发现该页并不含吞食线 / 异物的内容。此条为重要观点,但**来源待补**,待兽医核。〕
- 怀疑中毒伴突发呕吐:立即联系兽医 / 中毒热线;**不要自行催吐**。〔Cornell —— 突然成阵呕吐是猫中毒常见表现;除非兽医明确指示,否则不要催吐。https://www.vet.cornell.edu/.../poisons〕(注:v0.1"某些毒物催吐伤害更大"一句来源未支持,已删。)

### 🟡 在家观察 / 🟢 低风险的区分
- 🟢 健康猫偶尔呕吐(少于每月一次)、尤其吐出主要是毛,通常不异常。〔VCA —— 健康猫偶尔呕吐(少于每月一次)可能不代表异常,尤其吐出物主要是毛时。https://vcahospitals.com/know-your-pet/vomiting-in-cats〕🔍 本次核查未覆盖此 URL,待复核。
- 🟡 短期 / 偶发呕吐:持续不到 1–2 天、无其他异常 —— 可短时间在家观察。〔Merck 呕吐页 —— 短期或偶发呕吐一般不伴其他异常。〕
- 🟡 多数急性呕吐并非肠梗阻所致,常自限性(饮食不当、轻度肠胃炎、应激、晕车)。〔Merck 呕吐页〕🔍 此句本次核查抓取未覆盖,待人工核 Merck 原文。

### 居家护理(仅适用于可在家观察的情况)
- 急性呕吐:可短时间禁食让消化道休息,呕吐停止后先给少量水。⚠️**待兽医核:禁食时长对幼猫 / 已不吃东西的猫有肝脂沉积风险,草稿改为「先咨询兽医具体禁食多久」。**〔Merck 呕吐页 —— 原文为禁食 24 小时。〕
- 呕吐停止后喂少量、清淡、易消化食物。〔Cornell —— 持续性呕吐需支持性护理,呕吐停止后喂清淡易消化饮食。https://www.vet.cornell.edu/.../vomiting〕
- 给水少量多次。〔Merck 呕吐页〕

### 升级触发条件
- 观察期间呕吐持续不缓解、出现腹部不适、精神沉郁或发热。〔Merck 呕吐页〕🔍 此句本次核查抓取未覆盖,待人工核。
- 呕吐频率超过每周一次(慢性 / 反复)。〔Cornell —— 呕吐频率超过每周一次应及时由兽医评估。〕

### 常见误区
- 呕吐 ≠ 反流(regurgitation):反流是被动的、不需腹肌用力,意义不同。〔Merck 呕吐页〕
- 不要在无兽医指示下自行催吐。〔Cornell 中毒页〕(注:v0.1 另挂 VCA"不要给猫用人药"半句无 URL,已移至下方误区并标待补。)

## 1.2 腹泻

### 🔴 红旗
- 严重、带大量血的腹泻,或同时虚弱、发热、呕吐、腹痛、脱水、食欲废绝。〔VCA —— 严重血便和 / 或全身性疾病征兆应立即就诊。https://vcahospitals.com/know-your-pet/diarrhea-in-cats〕
- 黑色柏油样粪便(melaena)—— 可能提示肠道较高处出血。〔International Cat Care —— 黑色柏油样粪便可能提示肠道较高处出血,应联系兽医。https://icatcare.org/articles/diarrhoea-in-cats〕
- 腹泻持续超 1–2 天且伴全身性表现。〔Cornell —— 腹泻超一两天且伴食欲不振 / 嗜睡 / 呕吐应尽快就医。https://www.vet.cornell.edu/.../diarrhea〕
- 持续 / 反复呕吐 + 腹泻(无论有无血)。〔VCA emergencies 页〕

### 🟡 / 🟢 区分
- 腹泻新近、轻微、猫仍精神好能吃 —— 可对症处理观察。〔International Cat Care diarrhoea 页〕
- 突然换粮可致几天腹泻;应激(长途车程、寄养)可致短暂软便。〔Cornell diarrhea 页〕
- 粪便少量带血通常不需急诊但仍应兽医检查。⚠️**待兽医核:血量大小新手难判断,草稿建议保守 —— 任何便血先归「尽快就医」。**〔International Cat Care diarrhoea 页〕

### 居家护理
- 换简单、高度易消化饮食常有助控制腹泻;具体处方粮与兽医讨论。〔International Cat Care diarrhoea 页〕
- 兽医可能建议健康成年猫禁食至多 24 小时,或喂少量肠胃道处方粮。〔VCA diarrhea 页〕
- 注意补水,兽医可能建议合适电解质溶液。〔International Cat Care diarrhoea 页〕

### 升级触发条件
- 频繁稀便持续超 2 天。〔VCA diarrhea 页〕
- 家庭处理 4 天内无改善。〔VCA diarrhea 页〕
- 腹泻严重、伴其他不适、或持续超几天。〔International Cat Care diarrhoea 页〕

### 常见误区
- 腹泻不是病,是多种病的征兆。〔VCA diarrhea 页 / International Cat Care diarrhoea 页〕
- 不要自行给猫吃人用止泻 / 通便药 —— 人用泻药对宠物可能很危险。〔Merck —— 人用泻药对宠物可能非常危险。https://www.merckvetmanual.com/cat-owners/digestive-disorders-of-cats/disorders-of-the-stomach-and-intestines-in-cats;Cornell —— 部分非处方药对猫有害。〕

## 1.3 不吃东西 / 食欲废绝

### 🔴 红旗
- 完全不吃超过 2 天 —— 立即联系兽医。〔VCA —— 拒食超过两天立即咨询兽医。https://vcahospitals.com/know-your-pet/anorexia-in-cats〕
- 厌食短至 24 小时即可对成年猫健康造成严重影响。〔Cornell —— 厌食持续短至 24 小时即可严重影响成年猫健康。https://www.vet.cornell.edu/.../anorexia〕
- 突然不吃(尤其超重 / 肥胖猫)有肝衰竭风险 —— 肝脏脂肪沉积(脂肪肝)。〔VCA —— 突然食欲不振尤其超重 / 肥胖猫,因肝衰风险特别危险。https://vcahospitals.com/know-your-pet/anorexia-in-cats〕
- 进食不足即使短至 3 天也会使原有疾病恶化。〔International Cat Care —— 进食不足即使短至三天也会对猫产生显著负面后果。https://icatcare.org/resources/cat-carer-guide-managing-the-cat-that-wont-eat.pdf〕
- 不吃伴疑似疼痛(异常多睡 / 多睡在反常处、移动僵硬、梳毛与互动减少)—— 尽快就医。〔International Cat Care PDF —— 列出的疼痛迹象为:多睡且睡在反常处、移动与站立僵硬、排泄改变、梳毛及与家人互动减少。〕(注:v0.1"磨牙"一项及"伴呕吐/腹泻→尽快就医"的并列表述来源未支持,已收回。)

### 🟡 / 🟢 区分
- ⚠️**该症状基本没有「🟢 放着不管」的安全区间** —— 食欲减退与厌食都属需重视的疾病征兆。〔International Cat Care PDF〕
- 相对良性情形:刚就医 / 住院后应激、不喜欢新粮、家中新成员应激 —— 仍应告知兽医,不可默认无害。〔International Cat Care PDF;Cornell anorexia 页〕🔍 此条具体来源句本次核查未取到,待人工核。
- ⚠️**待兽医核:鉴于 24h / 3 天即有肝脂沉积风险,本症状分诊整体偏激进,不宜设较长「在家观察」窗口。**

### 居家护理(在配合兽医诊治期间鼓励进食)
- 保持冷静、维持平时喂食习惯;主人因焦虑围着食物打转本身会让猫更不想吃。〔International Cat Care PDF〕
- 先给猫以前爱吃、且适合病情的食物;新鲜、干净碗;先恢复食欲再过渡处方粮。〔International Cat Care PDF〕
- 少量多次(如每 2 小时一汤匙)。〔International Cat Care PDF〕
- 食物加热到接近体温可增香味;但若猫已对食物产生厌恶,则相反 —— 室温 / 偏凉气味弱、可能更易接受。〔International Cat Care PDF〕
- 用陶瓷 / 不锈钢碗(塑料碗沾气味);宽浅碗 / 平盘避免胡须碰碗壁。〔International Cat Care PDF〕
- 在安静、远离威胁源处喂食。〔International Cat Care PDF〕
- 充足、有吸引力的水源防脱水。〔International Cat Care PDF〕
- 不要把药混进正餐。〔International Cat Care PDF〕

### 升级触发条件
- 拒食超 2 天(若尚未就医)。〔VCA anorexia 页〕
- 出现反胃迹象:接近食盆又走开、咂嘴、夸张吞咽、大量流口水、见 / 闻食物干呕。〔International Cat Care PDF〕
- 居家护理无效或体重下降 —— 兽医可能需食欲刺激剂、止吐止痛药甚至饲管。〔International Cat Care PDF〕

### 常见误区
- 厌食绝不是猫「单纯决定不吃」。〔Cornell anorexia 页〕
- 不要强制 / 注射器灌食 —— 适得其反,增加吸入性肺炎风险、造成应激、可致习得性食物厌恶。〔VCA anorexia 页;International Cat Care PDF〕
- 区分真性厌食(不想吃)与假性厌食(想吃但因口腔疼痛 / 吞咽困难吃不了)—— 都需就医查因。〔VCA anorexia 页〕
- 不要在无兽医指示下给猫用人用药物。〔⚠️ 来源待补 —— v0.1 挂 VCA 但未给具体 URL,待补 / 待兽医核。〕

# 二、呼吸异常 / 出血 / 尿不出

> 这三个症状均偏急症,分诊基调整体保守:拿不准一律导向「立刻送医」。

## 2.1 呼吸异常(呼吸急促 / 费力 / 张口呼吸)

### 🔴 红旗
- 张口呼吸或喘气 —— 猫张口呼吸几乎总是不正常,立刻送医。〔VCA —— 呼吸困难尤其张口呼吸或喘气需立即就医。https://vcahospitals.com/know-your-pet/emergencies-in-cats〕
- 牙龈 / 嘴唇 / 鼻子发青 / 发紫(发绀)—— 氧气严重不足,必须立刻急诊。〔Cornell 哮喘页 —— 唇鼻变蓝即急症、兽医护理为必须。https://www.vet.cornell.edu/.../feline-asthma-risky-business-many-cats〕
- 安静休息 / 睡眠时呼吸频率持续 >30 次 / 分钟(正常 15–30)。〔VCA —— 安静休息 / 睡眠时正常 15–30 次 / 分钟;持续 >30 为异常。https://vcahospitals.com/know-your-pet/home-breathing-rate-evaluation〕
- 呼吸费力;猫蹲伏、低头、身体前伸来呼吸;哮喘发作时胸腹明显异常起伏。〔Cornell dyspnea 页 —— 呼吸困难的猫会低头、身体前伸。https://www.vet.cornell.edu/.../dyspnea-difficulty-breathing;"胸腹异常起伏"见 Cornell 哮喘页。https://www.vet.cornell.edu/.../feline-asthma-risky-business-many-cats〕
- 呼吸困难合并虚弱、咳嗽、行为改变 —— 按急诊。〔VCA home-breathing-rate 页〕
- 呼吸困难进展极快,初期被忽视会迅速恶化甚至死亡。〔Cornell dyspnea 页〕

### 🟡 / 🟢 区分
- ⚠️ **本节不设「在家观察」安全区间。** v0.1 原列「玩耍后短暂喘气可短暂观察」,经核查所引 Cornell dyspnea 页**不支持任何在家观察窗口** —— 该页通篇为「任何对猫呼吸能力的疑问都应立刻送医」。**故:猫的呼吸异常一律按急症,立刻送医。**〔Cornell dyspnea 页〕
- 区分「咳嗽」与「吐毛球」:哮喘猫颈部前伸蹲伏干咳,易被误认为呕吐 / 吐毛球。〔VCA —— 主人常误以为猫在呕吐或吐毛球,实为咳嗽。https://vcahospitals.com/know-your-pet/inhalant-treatment-for-feline-asthma-and-bronchitis〕(注:v0.1 误标此来源为"Merck",实为 VCA 自有页,已改。)
- 在家用「睡眠呼吸频率」客观判断:熟睡(无呼噜)时数 30 秒 ×2,持续 ≤30 相对安心,持续 >30 尽快联系兽医。〔VCA home-breathing-rate 页〕

### 居家护理
> 呼吸异常基本没有居家护理空间,核心是尽快、平稳送医。
- 送医途中尽量减少活动、保持安静。〔VCA emergencies 页〕
- 不要在家自行处理或喂药;只有兽医能做正确检查。〔Cornell dyspnea 页〕

### 升级触发条件
- 睡眠呼吸频率持续 >30 → 尽快联系兽医。〔VCA home-breathing-rate 页〕
- 张口呼吸 / 喘、唇鼻发青、呼吸费力 → 立刻急诊。〔VCA emergencies 页 + Cornell 哮喘页〕
- 已知哮喘的猫出现明显呼吸困难即急诊。〔Cornell 哮喘页〕

### 常见误区
- 误区:猫咳嗽 = 吐毛球 —— 哮喘 / 支气管炎干咳常被误判而延误。〔VCA inhalant-treatment 页〕(注:v0.1 误标"Merck",已改为 VCA。)
- 误区:哮喘是暹罗猫专属 —— 大型研究未证实。🔍 本次核查未能在所引 Cornell 哮喘页核到此句(抓取未覆盖品种段落),待人工核对。〔Cornell 哮喘页〕

## 2.2 出血

### 🔴 红旗
- 直接压迫 10–15 分钟仍止不住的外部出血 → 立刻急诊。〔VCA —— 10–15 分钟内止不住出血应送急诊。https://vcahospitals.com/know-your-pet/first-aid-for-bleeding-cats〕
- 出血合并牙龈苍白 / 发白、呼吸浅快或费力、虚弱 —— 可能内出血或休克。〔VCA first-aid-for-bleeding 页〕
- 呕血(鲜红血或「咖啡渣」样消化过的血)—— 猫呕血绝非正常。〔Merck —— 呕吐物可含鲜血或咖啡渣样消化血;溃疡猫可突发危及生命的消化道出血。https://www.merckvetmanual.com/cat-owners/digestive-disorders-of-cats/disorders-of-the-stomach-and-intestines-in-cats〕
- 黑色柏油样粪便(melena),尤其伴牙龈苍白。〔Merck 同上〕
- 怀疑误食抗凝血灭鼠药 —— 无论是否已出血都是急症,不要等出血症状。〔VCA —— 误食后 3–7 天才出现中毒征象,出血常发生在体内不可见处。https://vcahospitals.com/know-your-pet/rodenticide-warfarin-poisoning-in-cats〕
- 鼻出血止不住,或伴虚弱 / 呼吸困难 / 牙龈苍白 / 大量失血 / 虚脱。〔VCA —— 急救步骤无法止血或呼吸困难应立即就医。https://vcahospitals.com/know-your-pet/nose-bleeds-or-epistaxis-in-cats〕
- 胸 / 腹腔伤口有异物插着 → 不要拔出,直接送医。〔VCA first-aid-for-bleeding 页〕
- 有车祸 / 打架受伤史:即使外表正常也须兽医评估有无内伤。〔VCA emergencies 页〕

### 🟡 / 🟢 区分
- 浅表小划伤 / 轻微擦伤:可在家清洁后观察;但即便小伤口,「观察」也不等于「忽视」。⚠️「小伤口可能感染 / 形成脓肿」这一具体论断经核查不在所引 VCA care-of-open-wounds 页,该论断待补来源。〔VCA care-of-open-wounds 页〕
- 鼻出血后黑便 / 呕吐物带血块,不一定是消化道出血,可能是吞下鼻腔的血。〔VCA nose-bleeds 页〕

### 居家护理(适用于体表小伤口 / 已做止血)
- 控制出血首选直接压迫:干纱布盖伤口,再裹绷带 / 干净干布持续按压;可能时抬高患处高于心脏。〔VCA care-of-open-wounds 页〕
- 敷料被浸透时不要揭掉,直接再加一层。〔VCA first-aid-for-bleeding 页〕
- 爪子出血:包住持续加压,一般 5–10 分钟内停;断甲可用止血笔 / 硝酸银棒 / 止血粉(应急可用泡打粉、玉米淀粉、面粉)。〔VCA first-aid-for-bleeding 页 —— 原文应急替代物为 baking powder(泡打粉);v0.1 误作"小苏打",已改。〕
- 清洁伤口只用温自来水或温生理盐水。〔VCA care-of-open-wounds 页〕
- 不要往伤口用肥皂、酒精、双氧水、茶树油 / 草药,不要自行涂药膏(除非兽医指示)。〔VCA care-of-open-wounds 页〕
- 鼻出血:保持猫安静,鼻梁放冰袋;不要往鼻孔塞棉花(会引发打喷嚏加重出血)。〔VCA nose-bleeds 页〕
- 处理受伤的猫注意自身安全:受伤疼痛的猫即使平时乖也可能抓咬。〔VCA first-aid-for-bleeding 页〕

### 升级触发条件
- 体表出血压迫 10–15 分钟止不住 → 立刻急诊。〔VCA first-aid-for-bleeding 页〕
- 出现牙龈苍白 / 呼吸浅快 / 虚弱 → 怀疑内出血 / 休克,立刻急诊。〔VCA first-aid-for-bleeding 页〕
- 在家观察的伤口分泌物连续数日呈血性 / 绿色 / 黄色 → 联系兽医。〔VCA care-of-open-wounds 页〕
- 呕血、黑色柏油便、不明瘀斑 / 肿胀 → 立刻就医(尤其有鼠药接触可能时)。〔VCA rodenticide 页〕

### 常见误区
- 误区:外表没事 = 没受伤 —— 车祸 / 打架后可能有看不见的内伤。〔VCA emergencies 页〕
- 误区:误食鼠药「等出血了再去」—— 抗凝血鼠药出血延迟 3–7 天且多在体内,应怀疑误食时立刻就医。〔VCA rodenticide 页〕
- 误区:用双氧水 / 酒精给猫伤口消毒 —— 权威页明确不建议。〔VCA care-of-open-wounds 页〕

## 2.3 尿不出 / 排尿困难(尿道阻塞)

> ⚠️ **完全性尿道阻塞(几乎只发生在公猫)是数小时到数天内可致命的真正急症。** 任何「频繁进猫砂盆却尿不出」都应按急诊处理。

### 🔴 红旗
- 反复使劲排尿却完全尿不出、或只有几滴。〔Cornell —— 阻塞的猫使劲排尿、频繁尝试、产尿极少或无。https://www.vet.cornell.edu/.../feline-lower-urinary-tract-disease〕
- 频繁跑猫砂盆但不产尿 / 只产小滴 → 立刻就医。〔VCA —— 频繁去猫砂盆不产尿或只产小滴,立即就医。https://vcahospitals.com/know-your-pet/emergencies-in-cats〕
- 在猫砂盆里嚎叫 / 因疼痛大叫;随时间越来越痛苦。〔Merck —— 「在猫砂盆里嚎叫」列为需立即就医的危急症状。https://www.merckvetmanual.com/cat-owners/kidney-and-urinary-tract-disorders-of-cats/noninfectious-diseases-of-the-urinary-system-of-cats〕
- 出现精神沉郁、食欲废绝、不愿活动、呕吐、脱水 —— 阻塞已致毒素蓄积,病情快速恶化。〔Merck 同上〕
- 时间窗:完全阻塞不解除,可在约 2–3 天内死亡。〔Merck 同上〕第二来源印证:从完全阻塞到死亡可能不足 24–48 小时。〔Cornell FLUTD 页〕
- 公猫(含绝育公猫)风险显著更高(尿道更长更窄)。〔Cornell FLUTD 页〕
- 阻塞致电解质紊乱可引发致命心律失常。〔Merck 同上〕

### 🟡 / 🟢 区分
- 关键区分 **阻塞 vs 未阻塞**:未阻塞(膀胱炎 / FLUTD)使劲后仍能排少量尿;阻塞使劲后几乎无尿。🔍 此条引 icatcare.org,本次核查 403 无法打开,需人工 / 兽医核。〔International Cat Care FLUTD 页 https://icatcare.org/articles/feline-lower-urinary-tract-disease〕
- 关键区分 **尿道阻塞 vs 便秘**:两者都表现为在猫砂盆里使劲,极易混淆。⚠️ 经核查,所引 Petplan 页给的是**姿势**区分(便秘时背更弓 / 阻塞时身体更蜷缩、可能叫唤),并非 v0.1 所写的「反复使劲更常是阻塞」;且 Petplan 是宠物保险机构科普页(经 MRCVS 兽医审阅,非一手兽医组织)。**产品决策:拿不准时一律按尿道阻塞这一急症处理(偏保守)。此鉴别须兽医用一手来源重写。**〔Petplan https://www.petplan.co.uk/cat-insurance/cat-care-and-advice/cat-constipation-and-urinary-blockages.html〕
- 相对低风险:排尿频率增多、少量血尿、异常地点排尿但仍能正常产尿 —— 多见于未阻塞 FLUTD,仍需就医但通常非「数小时致命」级。〔Cornell FLUTD 页〕

### 居家护理
> ⚠️ 尿道阻塞**没有有效居家护理**,唯一正确动作是立刻送急诊。不要在家「挤膀胱」。
- 怀疑尿道阻塞,唯一应做的是立即联系兽医 / 急诊。🔍 此条引 icatcare.org,本次核查 403 无法打开,需人工 / 兽医核。〔International Cat Care urethral-obstruction 页 https://icatcare.org/articles/urethral-obstruction-in-cats〕

### 升级触发条件
- 任何「频繁进猫砂盆却尿不出 / 只尿几滴」→ 立刻急诊。〔VCA emergencies 页〕
- 排尿困难的猫出现精神沉郁 / 不吃 / 呕吐 → 立刻急诊。〔Merck 同上〕
- 公猫出现持续使劲排尿、嚎叫、舔舐生殖器、反复跑猫砂盆 → 按尿道阻塞急症处理。〔Cornell FLUTD 页〕

### 常见误区
- 误区:把「使劲尿不出」当便秘在家等会延误 —— 误判方向风险高。⚠️「会致命」的措辞非所引 Petplan 页原文;该方向在 Cornell / Merck 一手来源有依据,但此条须改挂一手兽医来源。〔待补一手来源〕
- 误区:母猫也常发生致命尿道阻塞 —— 完全性阻塞几乎只见于公猫;但母猫排尿异常仍需就医。〔VCA —— 尿道阻塞几乎只发生于公猫。https://vcahospitals.com/know-your-pet/cystitis-and-lower-urinary-tract-disease-in-cats〕
- 误区:尿带血就一定是尿道阻塞 —— 血尿可由多种原因引起,需就医鉴别。⚠️「血尿原因多样」「难尿+频繁+血尿组合提示致命疾病」这两句具体表述经核查未在所引 VCA cystitis 页找到对应原文,待兽医核 / 待补来源。〔VCA cystitis 页〕

# 三、精神差 / 误食中毒 / 通用红旗

## 3.1 精神差 / 嗜睡 / 躲起来不动

### 🔴 红旗
- 突然瘫倒、站不起来、叫名无反应、或失去意识。〔Anicira —— Collapse:猫瘫倒站不起来必须立刻联系兽医,可能心 / 肺 / 脑 / 循环出问题。https://anicira.org/resources/10-signs-your-cat-is-having-a-veterinary-emergency/〕
- 精神差伴牙龈 / 黏膜发白、严重萎靡、四肢和耳朵发凉 —— 休克表现。〔VCA —— 休克征象:黏膜苍白发白、严重精神沉郁、四肢发凉。https://vcahospitals.com/know-your-pet/emergencies-in-cats〕
- 精神差 / 虚弱伴呼吸急促 / 费力 / 张口呼吸。〔VCA emergencies 页〕
- 精神差伴牙龈发白 / 发黄 / 发紫(任一异常颜色)—— 不论行为如何按急症。〔PetMD —— 见到白 / 黄 / 蓝色牙龈,无论猫行为如何都按急症。https://www.petmd.com/cat/symptom/pale-gums-in-cats〕
- 精神差突然发生、程度严重,或伴近期可疑接触毒物。〔Cornell —— 「迟钝 / 精神差」是中毒常见征象之一。https://www.vet.cornell.edu/.../poisons〕

### 🟡 / 🟢 区分
- 🟡 轻度精神差 / 轻微食欲变化也应尽早就诊评估(猫极擅长隐藏疾病)。⚠️ v0.1「24–48 小时内就诊」这个具体时间窗经核查不在所引 VCA 页,已删除。〔VCA —— 猫进化出隐藏疾病与疼痛的本能。https://vcahospitals.com/know-your-pet/recognizing-signs-of-illness-in-cats〕
- 🟡 精神差伴食欲 / 饮水量明显改变,值得重视。〔VCA recognizing-signs 页〕
- 🟡 精神差伴不梳毛、被毛变乱变油 —— 「感觉不舒服」信号。〔VCA recognizing-signs 页〕
- 🟢 相对低风险:活力下降 / 躲藏也可能由环境应激引起,排除疾病后风险较低 —— 但仍建议先由兽医排除身体问题。⚠️ v0.1 列的具体应激诱因清单(装修 / 搬家 / 新生儿 / 失去同伴)经核查不在所引 VCA 页,已删除。〔VCA recognizing-signs 页〕

### 居家护理 / 观察要点
- 留意这些方面相对自身基线的变化:是否吃、是否喝、用猫砂盆情况、活动量、对呼唤的反应。⚠️ v0.1「逐项记录下来作为就诊信息」这个动作建议是草稿补充,所引 VCA 页只描述这些维度的异常、未给「记录」建议。〔VCA recognizing-signs 页〕

### 升级触发条件
- 拒食超过 24 小时,立刻就医。〔VCA —— 若猫 24 小时未正常进食,立即就医。https://vcahospitals.com/know-your-pet/recognizing-signs-of-illness-in-cats〕
- 精神差加重为虚弱 / 无法站立 / 无反应,或出现呼吸困难 → 立刻急诊。〔VCA emergencies 页〕

### 常见误区
- 误区:猫只是没精神在睡 —— 真正的精神差不是「困」,是相对自身基线的能量与行为下降。〔VCA recognizing-signs 页〕
- 误区:躲起来肯定只是怕生 —— 躲藏既可能应激也可能疼痛 / 疾病,需兽医鉴别。〔VCA recognizing-signs 页〕

## 3.2 可能误食 / 中毒

### 🔴 立刻送医
- 怀疑或确知吃了任何有毒物质(防冻液、木糖醇、巧克力、灭鼠药等)→ 立刻就医。〔AVMA —— 「怀疑或确知宠物吃了有毒物」列为须立即就医急症。https://www.avma.org/resources/pet-owners/emergencycare/13-animal-emergencies-require-immediate-veterinary-consultation-andor-care〕
- 接触 / 误食百合(Lilium / Hemerocallis)任何部分(含花粉、瓶水)→ 立刻就医,可致命急性肾衰。〔VCA —— 真百合极少量(花瓣 / 叶 / 花粉 / 瓶水)即可致严重、可能不可逆的肾衰。https://vcahospitals.com/know-your-pet/household-hazards-toxic-hazards-for-cats〕
- 误食对乙酰氨基酚(扑热息痛 / 泰诺)—— 哪怕一片 —— 立刻就医,可致命。〔VCA household-hazards 页〕
- 出现中毒征象(精神迟钝、步态不稳、流口水、呼吸沉重、腹泻、抽搐、突然呕吐)→ 立刻就医。〔Cornell poisons 页〕

### 常见有毒「食物」
- 巧克力 / 咖啡 / 含咖啡因食物(甲基黄嘌呤)—— 可致呕吐腹泻、喘、过度饮水排尿、心律异常、震颤、抽搐甚至死亡;黑巧风险更高。〔ASPCA —— https://www.aspca.org/pet-care/aspca-poison-control/people-foods-avoid-feeding-your-pets〕中毒征象通常误食后约 6–12 小时内出现。〔Merck —— 巧克力中毒征象 6–12 小时内出现。https://www.merckvetmanual.com/toxicology/food-hazards/chocolate-toxicosis-in-animals〕
- 洋葱 / 大蒜 / 香葱(葱属,生熟均)—— 破坏红细胞致贫血,猫比狗更敏感。〔ASPCA people-foods 页 —— 该条列 onion / garlic / chives;韭葱(leeks)同属葱属、亦有毒,但不在此 ASPCA 条目内。〕
- 葡萄 / 葡萄干 —— 与肾损伤相关。⚠️ ASPCA 此条原文整段讲的是**狗**(狗无法代谢酒石酸);对猫是否同等风险该来源未说明,**待兽医核 / 待补针对猫的来源**。〔ASPCA people-foods 页〕
- 酒精及生酵母面团 —— 酒精致呕吐腹泻、共济失调、呼吸困难、震颤;生面团发酵产酒精并致胃胀。〔ASPCA people-foods 页〕
- 木糖醇(无糖口香糖等)—— 致低血糖、可能肝损伤,可进展为抽搐。〔ASPCA people-foods 页〕
- 食盐 / 过咸食物 —— 过量致呕吐腹泻、震颤、抽搐甚至死亡。〔ASPCA people-foods 页〕
- 生肉 / 生蛋 —— 可能含沙门氏菌、大肠杆菌。〔ASPCA people-foods 页〕

### 常见有毒「植物」
- 真百合(Lilium / Hemerocallis)—— 对猫极度危险,极少量即可致严重可能不可逆肾衰。〔ASPCA —— https://www.aspca.org/news/top-10-toxic-plants-pets-what-look-out〕
- 杜鹃花 / 山踯躅(Azalea / Rhododendron)—— 大量误食可致心律不齐、抽搐。〔ASPCA top-10 页〕
- 苏铁 / 铁树(Sago Palm)—— 种子最危险,可致肝衰甚至死亡。〔ASPCA top-10 页〕
- 铃兰 —— 含强心苷,致呕吐腹泻和心律异常。〔ASPCA —— 经核查正文已确认。https://www.aspca.org/news/which-lilies-are-toxic-pets〕
- 白鹤芋(Peace Lily)、马蹄莲 —— 含不溶性草酸钙,致口腔疼痛、流口水、肠胃不适。〔VCA household-hazards 页〕
- 郁金香 / 水仙 —— 致肠胃不适;吞大块球茎可能致梗阻。〔ASPCA top-10 页〕
- Cornell 也把百合、郁金香、毛地黄、蔓绿绒列为对猫危险植物。〔Cornell poisons 页〕

### 常见有毒「人药 / 家居物」
- 对乙酰氨基酚 —— 猫极度敏感(葡萄糖醛酸转移酶活性低);可数小时内致高铁血红蛋白血症,征象含嗜睡、虚弱、呕吐、低体温、面 / 爪水肿、发绀、呼吸困难。〔Merck —— https://www.merckvetmanual.com/toxicology/toxicoses-from-human-analgesics/toxicoses-from-human-analgesics-in-animals〕
- 布洛芬 / 萘普生 —— 对猫有毒,可致肾衰和胃溃疡;猫对其敏感、中毒剂量低于狗。〔VCA household-hazards 页;Merck human-analgesics 页〕(注:v0.1"对猫属最致命药物之列"为程度词夸大,已收回。)
- 阿司匹林 —— 猫极易中毒,半衰期长达约 37.5 小时。〔Merck human-analgesics 页〕
- 抗抑郁药 / ADHD 药 —— 致心血管和神经系统过度兴奋。〔VCA household-hazards 页〕
- 含拟除虫菊酯的「狗用」驱蚤产品(permethrin)—— 对猫高度有毒。〔VCA household-hazards 页〕
- 维生素 D3、含 α-硫辛酸的人用补充剂 —— D3 致危及生命的高血钙;ALA 致低血糖、抽搐、肝损伤。〔VCA household-hazards 页〕
- 茶树等浓缩精油 —— 对猫有毒,可致精神抑制、低体温、肝损伤。⚠️ v0.1「一滴即可致中毒」的具体剂量经核查不在所引 VCA 页,已收回。〔VCA household-hazards 页〕
- 家用清洁剂 —— 致肠胃不适、化学灼伤、呼吸道症状等。〔VCA household-hazards 页〕灭鼠药、含锌物(1982 年后美分)、铅 —— 锌 / 铅可致致命贫血与肾衰。〔Cornell poisons 页 —— 吞一枚硬币可致致命贫血与肾衰。〕(注:v0.1 把灭鼠药 / 锌 / 铅一并挂在 VCA 名下,VCA 页只点名清洁剂,已分拆来源。)

### 误食后「该做」与「不该做」
- 该做:立刻联系兽医 / 中毒控制中心。〔Cornell poisons 页〕(尽量同时说清何时、误食何物、量、猫现状 —— 此为操作建议,非来源原文。)
- 该做:若毒物沾在猫的皮肤 / 被毛 / 爪上,阻止猫继续舔舐(并尽快就医)。〔⚠️ **来源对不上** —— 经核查所引 VCA 页并不含此条;此条为合理观点,但**来源待补**,待兽医核。〕
- 不该做:不要自行催吐(除非兽医指示)。〔Cornell poisons 页〕
- 不该做:不要用双氧水给猫催吐 —— 即使少量也损伤口腔咽喉胃。〔VCA household-hazards 页〕
- 不该做:不要给猫用任何人用止痛药(对乙酰氨基酚、布洛芬、阿司匹林等)。〔据 Merck human-analgesics 页详述的毒性归纳。〕

### 升级条件
- 即使猫暂时看似没事,只要确知误食高危物(百合、对乙酰氨基酚、布洛芬、灭鼠药、防冻液)立即就医 —— 百合肾损伤可数小时内开始,巧克力征象多 6–12 小时后显现。〔Merck chocolate 页;ASPCA top-10 页〕
- 误食后出现抽搐 / 震颤、步态不稳、呼吸困难、牙龈变色、虚弱瘫倒任一 → 立刻急诊。〔Cornell poisons 页;AVMA 13-emergencies 页〕

## 3.3 跨症状「立刻送医」通用红旗

- 瘫倒、站不起来。〔Anicira 10-signs 页〕
- 意识不清 / 失去意识 / 昏倒。〔VCA emergencies 页;AVMA 13-emergencies 页〕
- 抽搐:单次发作超过 5 分钟,或短时间内连续多次发作。〔VCA emergencies 页〕⚠️ 来源不一致:PetMD seizures 页给出更保守的 2–3 分钟阈值 —— 经核查两个数字均属实,是来源本身分歧,须兽医统一。
- 持续性癫痫状态(多次发作之间不恢复)。〔PetMD seizures 页 https://www.petmd.com/cat/symptoms/seizures-cats〕
- 牙龈 / 黏膜发白 / 发黄 / 发紫(任一异常颜色)。〔PetMD pale-gums 页〕
- 呼吸困难:又快又费力、张口呼吸 / 喘、腹部用力起伏。〔VCA emergencies 页;AVMA 13-emergencies 页〕
- 休克征象:黏膜苍白发白、严重萎靡、四肢和耳朵发凉。〔VCA emergencies 页〕
- 严重出血:大量出血,或 5 分钟内止不住;鼻 / 口 / 肛门出血、咳血、尿血。〔AVMA 13-emergencies 页〕
- 严重外伤 / 头部外伤 / 穿透伤 / 眼部受伤(坠楼、车祸)。〔VCA emergencies 页〕
- 骨折 / 严重跛行 / 腿无法承重或移动。〔AVMA 13-emergencies 页;Anicira 10-signs 页〕
- 后腿突然拖行、瘫软并大声嚎叫(可能血栓)。〔Anicira 10-signs 页〕
- 无法排尿 / 排便,或排尿排便明显疼痛(公猫努责排不出尿 = 尿道堵塞)。〔Anicira 10-signs 页;AVMA 13-emergencies 页〕
- 严重疼痛 / 极度焦虑不安。〔AVMA 13-emergencies 页;VCA emergencies 页〕
- 热应激 / 中暑。〔AVMA 13-emergencies 页〕
- 严重呕吐或腹泻:24 小时内超两次,或合并明显不适。〔AVMA 13-emergencies 页〕
- 拒绝饮水达 24 小时或更久。〔AVMA 13-emergencies 页〕
- 确知或怀疑误食有毒物质。〔AVMA 13-emergencies 页〕
- ⚠️**「体温过低」未作为独立红旗写入** —— 未找到把低体温单列为通用急症的权威兽医页;它只作为「休克」相关征象(四肢发凉)或某些中毒征象出现。是否单列待兽医定夺。
