## 一、核心医学原则(可直接用于报告)

- **「行为问题应先排除医学原因」是兽医共识。**(Merck 原文,反复出现)
  > _"medical conditions can also lead to aggression. This is why veterinarians rule out medical causes of aggression."_
  > _"With housesoiling, your veterinarian will always first exclude any medical problems."_
  > _"Hyperesthesia... Your veterinarian will first rule out medical causes, especially pain and skin conditions."_
  → 反复强调:**任何「突然」/「持续」的行为变化,先看兽医做全身体检 + 血尿检,再考虑行为问题。**

- **触碰激发的攻击 = 疼痛攻击(pain aggression)**(Merck)
  > _"Pain aggression is a defensive reaction that occurs when a cat is in pain. It may happen when a cat anticipates being moved or touched."_
  → 之前温顺、现在被摸某部位就嚎叫 / 咬 / 抓 → 那个部位很可能在疼。常见来源:关节(尤其老猫)、口腔、外伤、肿瘤。

- **老年猫(>10 岁)行为变化要排除 CDS / 甲亢 / 高血压。**(Merck)
  > _"Older cats may also have cognitive dysfunction syndrome, which is similar to Alzheimer disease and senile dementia in people and can cause disorientation, agitation, anxiety, memory loss, housesoiling, and personality changes."_
  → 老年猫**白天睡多、晚上叫、迷失方向、性格改变**是 CDS 典型表现;不是「老了脾气差」而已。

- **App 不诊断 / 不开行为处方;医学排除 → 普通兽医;行为治疗 → 委婉建议看行为咨询师 / board-certified veterinary behaviorist。**

## 二、主要类型(用于追问 + 用户认知)

### A. 攻击性(aggression)
Merck 列出 10+ 种类型,核心要点:
- **Pain aggression** —— 触碰激发,常见疾病诱发
- **Fear aggression** —— 害怕逃不掉时反击
- **Status-related / "leave-me-alone bite"** —— 不像狗,跟资源无关,是猫的独有类型
- **Redirected aggression** —— 看到窗外的猫激动后攻击家里另一只
- **Maternal aggression** —— 母猫保护小猫(断奶后通常消退)
- **Predatory / Play aggression** —— 跟玩耍 / 狩猎本能有关
- 警告信号:**瞳孔放大、尾巴抽动、耳朵向后** —— Merck 明确这是攻击前兆

### B. 排泄异常(elimination problems)
Merck 指出:**这是猫最常见的行为问题。**
- **Spraying(尿喷洒)** —— 站着、尾巴竖立颤抖,垂直面;**绝育能消除 / 减少 90%**(Merck 数据)
- **Inappropriate elimination** —— 在水平面(地毯 / 床 / 衣物)上排尿排便;可能是 substrate preference / location preference / litter box aversion
- **关键:兽医必须先排除泌尿系统疾病、肠道病、糖尿病、肾病 —— 这些都可能让猫"乱排泄"**(本 App 已有 `pee` 症状轨道覆盖泌尿急症;`behavior` 轨道覆盖"已排除医学原因后的行为问题")

### C. 过度发声(excessive vocalization)
Merck + 搜索结果共识:
- 注意力寻求、性行为(发情)、玩耍、医学问题、不适、攻击
- **中老年猫的过度发声** → 兽医重点排除**疼痛、内分泌(尤其甲亢)、高血压、CDS**

### D. 强迫行为(compulsive)
Merck:
- **过度梳理 / 舔毛**(常被误以为皮肤病)
- **咬毛 / 布 / 塑料 / 橡胶 / 纸板 / 线**
- 暹罗及类似品种**遗传倾向**咬布
- 主要诱因是**应激 / 焦虑**
- **兽医先排除疼痛、皮肤病、内分泌病**,再当行为问题处理

### E. 过敏感综合征(hyperesthesia syndrome)
Merck:
- 触碰背部即嚎叫 / 躁动 / 攻击 / 过度梳理 / 嘶叫咬背 / 跑掉 / 排便
- **兽医先排除疼痛和皮肤病**
- 排除后才考虑强迫障碍

### F. 老年猫认知功能障碍(CDS)
Merck:类似阿尔茨海默 / 老年痴呆
- 迷失方向、躁动、焦虑、记忆丧失、乱排泄、性格改变
- 「**白天睡多 + 晚上叫 + 在家里走丢**」是典型组合
- 不可逆但可以管理(环境调整 + 药物)

## 三、什么时候算「立刻就医」(red 路径)

汇总 Merck + 临床常识:

- **碰某个部位就嚎叫 / 攻击 + 之前温顺** —— pain aggression,该部位很可能在疼
- **行为变化 + 联合其它躯体症状**(不吃 / 吐 / 拉 / 喘 / 走路不稳 / 流口水) —— 提示全身性问题
- **突然走路不稳 / 转圈 / 撞东西 / 意识异常** —— 神经急症
- **完全拒绝进食 + 躲起来不动 + 看着痛苦** —— 综合性恶化信号(跟 lethargy 红档同源)

## 四、什么时候是「黄档:这周内约门诊」

- **任何「突然」「持续」的行为变化** —— Merck 反复强调先做医学排除
- **老年猫(>10 岁)的行为变化** —— 重点排除甲亢 / 高血压 / CDS / 慢性疼痛
- **乱排泄但能排尿** —— 先做尿检血检排除医学原因,再考虑行为路径
- **过度梳理 / 强迫行为持续** —— 先排除皮肤病、疼痛、内分泌病
- **过敏感综合征样表现** —— 先排除疼痛、皮肤、神经

## 五、什么算「绿档:正常行为差异」

注意 —— App 必须**先排除上面 red / yellow 信号**才能把行为变化归绿:

- **母猫发情**(未绝育)—— 吼叫、在地上打滚、抬尾巴、不寻常温顺;**这是生理过程,不是病**;绝育可消除
- **公猫青春期到性成熟**(6-9 个月)—— 喷尿、领地标记;**绝育可消除约 90%**
- **应激短期反应** —— 搬家、新成员、装修、新猫狗。Merck 明确:几天到一两周内逐渐适应;期间提供「房间内独享资源」(食物、水、猫砂盆、藏身处)
- **品种特征** —— 暹罗黏人多话、布偶慢热、英短淡然 —— 个性差异不是病
- **幼猫(2-7 月)疯狂玩耍 / 半夜跑酷** —— 高能高活动期,正常发育阶段
- **季节性变化** —— 春秋发情季活跃度上升;冬季睡得多

## 六、主人能做的(在医学排除前后)

### 排除医学原因前
- **观察并记录**:什么时候开始、什么场景下出现、有没有触发因素
- **拍视频**给兽医看,猫到了医院可能不表现
- **不要惩罚** —— Merck 明确反对,会让猫怕主人 + 加重问题
- **保护自己** —— 攻击性猫先隔离到独立房间,避免被咬伤(猫咬抓伤易感染)

### 医学排除后的常见环境调整(Merck 给的方向)
- 提供多个**资源点**(食物、水、猫砂盆),按"猫数 + 1"配置
- **垂直空间**(猫爬架、窗台)— 安全感来自高处
- **每日 2 次 ×10 分钟逗猫棒互动**(模拟狩猎)
- 喂食器小份多次、漏食球(模拟狩猎 + 防无聊)
- 必要时**费洛蒙喷剂 / 扩散器**(Feliway 等)— 兽医建议
- 严重 / 顽固的 → 推荐 **board-certified veterinary behaviorist**(国内可能没有此证书 → 找资质背景的动物行为咨询师 + 兽医联合处理)

### 主人**不能**做
- **不要打猫 / 体罚 / 用喷水恐吓** —— Merck 明确反对,会破坏信任
- **不要给猫吃人的镇静药 / 抗焦虑药** —— 剂量错误可致命
- **不要忽视「只是脾气坏」** —— 突然变得脾气坏的猫,9 成是身体出了问题

## 七、对接产品的具体改造

### 加新症状 `behavior`(完整轨道)

1. **`SYMPTOM_LABELS`** 加 `behavior: "行为突变"`
2. **`symptoms/page.tsx`** 加卡片(tier: `"common"`)
3. **`triage.ts`** 加 `behaviorFlow` 3 题:
   - Q1 主要变化(攻击 / 躲藏 / 过度发声 / 乱排 / 不爱玩 / 稍微敏感)
   - Q2 多选伴随:**触碰激发攻击 / 嚎叫 redFlag** + **联合躯体症状 redFlag**(不吃 / 吐 / 拉 / 喘)+ **神经症状 redFlag**(走路不稳 / 转圈 / 撞东西)+ 老年猫白天睡多晚上叫迷失方向(CDS)+ 母猫发情(w0)+ 最近大变化(应激,w1)+ 都没有
   - Q3 时长(刚一两天 / 一两周 / 超过两周或反复)
4. **`report/page.tsx`** 加 `behavior` 组三档:
   - **red:** pain aggression / 联合躯体症状 / 神经症状 —— 这是疾病的间接表现,立刻就医
   - **yellow:** 突然 / 持续行为变化 —— 这周内约门诊,兽医先做体检 + 血尿检排除医学原因,然后再考虑行为咨询;老年猫优先排除 CDS / 甲亢 / 高血压
   - **green:** 发情、应激短期反应、品种特征、幼猫高能期、季节性变化 —— 提示环境调整 + 关注变化
5. **`NO_GREEN_GROUPS` 不加 behavior** —— 行为变化有真实的"生理性"/"环境性"原因(发情、应激、品种)

### 文案要点

- **「猫的脾气不会无缘无故变」** —— 给新手家长直观的提示
- **「触碰激发攻击 = 那里在疼」** —— Merck 的关键医学共识
- **「先看普通兽医做体检血尿检 → 再考虑行为问题」** —— 反复出现的医学原则
- **「老年猫的『脾气怪』可能是 CDS / 甲亢」** —— 老年人最常忽视的方向
- **「绝育能解决 90% 喷尿和大部分公猫发情期攻击」** —— 实证有效的具体建议
- **「应激反应几天到一两周适应」** —— 给主人耐心的依据
- **「不要惩罚」** —— Merck 明确反对,反着说是好的提示

## 八、待执业兽医核对

1. **国内是否有 board-certified veterinary behaviorist 体系** —— 没有的话,App 该怎么命名转诊路径(动物行为咨询师?有资质的兽医?)。
2. **Feliway / 费洛蒙扩散器** 国内可及性 + 实际效果证据等级 —— 是否要在 App 文案里点名某种产品。
3. **「猫咬抓伤易感染」+ 猫抓病(Bartonella henselae) + 狂犬病暴露处理** —— 是否在攻击性 red 文案里要明确「人被咬抓伤后该立刻怎么做」(目前不在产品红线,但被咬抓的主人是真实场景)。
4. **CDS 在国内宠物医院的诊断 + 用药(司来吉兰等)的可及性** —— 是否要在 yellow 文案里点名。
5. **老年猫的「行为变化 → 甲亢 / 高血压」筛查** —— 国内宠物医院常规检测可及性 / 价格区间。
