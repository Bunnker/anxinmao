## 一、关键事实

- **60-90% 的老年猫有骨关节炎(OA),但常被忽视。**(Merck 原文)
  > _"The condition is common in cats but it may not be noticed because cats often hide signs of pain. Approximately 60% to 90% of older cats have osteoarthritis."_
  → **猫善于藏疼痛 + OA 进展慢** → 主人常以为「老了不爱动」,实际是关节疼。

- **主动脉血栓栓塞(ATE / FATE)是真急症**,**数小时窗口**。
  典型表现:**突然后腿瘫软 + 大叫疼痛 + 后腿冰凉 / 摸不到股动脉脉搏 + 后腿垫发紫 / 苍白**。常见于已有 HCM(肥厚性心肌病)的猫(很多 HCM 在 ATE 发作前没有任何征兆)。**死亡率高、致残率高。**

- **创伤是急性跛行最常见原因。** 高处坠落、车祸、被打架、被门夹尾巴或腿、被踩到 —— 可能骨折 / 关节脱位 / 韧带断裂 / 软组织伤。

- **关节脱位 / 骨折 = 急症。** 不要在家「正位」—— 可能加重损伤、感染、血管神经压迫。

- **败血性关节炎(septic arthritis)** —— 细菌从血源 / 外伤伤口 / 手术后进入关节,伴**发热、嗜睡、食欲下降** + 关节肿痛。属急症方向。

- **免疫介导性多关节炎(immune-mediated polyarthritis)** —— 多关节同时痛、伴发热、反复发作。需兽医诊断(关节穿刺 + 血检)。

- **诊断必须由兽医做体检 + X 光,有时需要 CT / MRI / 关节穿刺。** App 不诊断。

- **App 不能开任何止痛药。** 特别警告:**对乙酰氨基酚(扑热息痛 / 泰诺)** 哪怕一片可对猫致命(v0.2 §3.2 已点名);**布洛芬 / 萘普生 / 阿司匹林** 都对猫剧毒。

## 二、典型表现 + 鉴别

### A. 急性跛行(创伤、ATE、骨折、脱位)
- **完全不能用某条腿、提着走** —— 骨折 / 严重软组织伤 / 关节脱位
- **腿部肿胀、变形、有开放伤口** —— 骨折 / 严重外伤
- **突然后腿瘫软 + 大叫 + 后腿冰凉 / 垫发紫** —— ATE,真急症
- **刚摔过 / 车祸 / 打架后跛行** —— 默认按外伤就医,即使外表正常

### B. 慢性跛行(OA、髋发育不良、髌骨脱位)
- **持续 / 反复一瘸一拐,慢性**
- **跳不上以前能跳的地方、犹豫上下台阶**
- **关节僵硬 / 起身慢 / 不爱活动**
- **梳毛不到的部位毛糙凌乱**(因为转身梳毛会痛)
- **老年猫躲在低处不爱上下跳**

→ Merck 强调:这些**新手家长常误判为「老了正常」,实际是 OA 在疼**。

### C. 系统性 / 感染相关
- **跛行 + 发热 + 不吃 + 萎靡** —— 败血性关节炎 / 免疫性多关节炎(发热 + 多关节痛)
- **跛行后出现皮疹 + 系统症状** —— 免疫性,如 SLE
- **关节肿胀 + 拒触** —— 任何关节炎症

### D. 局部问题
- **肉垫里有刺 / 玻璃 / 草籽** —— 看得到的异物
- **脚指甲断了 / 嵌入肉垫 / 流血** —— 常见但易忽视
- **被门夹后**

## 三、何时算「立刻就医」(red 路径)

汇总 Merck + 临床:

- **后腿突然瘫软 + 大叫 + 后腿冰凉 / 摸不到脉搏 / 垫发紫**(ATE)— **几小时窗口**,真急症
- **完全不能用某条腿、提着走**(怀疑骨折 / 严重脱位)
- **腿部肿胀、明显畸形、有开放伤口**
- **刚摔过 / 车祸 / 打架后跛行** —— 即使看上去能走也要排除内伤
- **跛行 + 发热 / 不吃 / 萎靡**(怀疑败血性关节炎 / 全身性)
- **多关节同时痛 + 系统症状**(免疫性多关节炎可能)

---

## 四、何时算「黄档:这周内约门诊」

- **持续跛行 > 几天**,无急症信号
- **老年猫(> 10 岁)+ 跳跃减少 / 起身慢 / 不爱活动** —— OA 高度可能,值得评估
- **肉垫卡异物 / 指甲问题** —— 多数可门诊取出 / 处理
- **运动后 / 跳跃后偶尔瘸一下、几小时后好但反复** —— 软组织慢性伤可能

---

## 五、什么算「绿档:轻微观察」

- **偶尔瘸一下、几分钟到几小时就好** —— 常是软组织小拉伤;限制活动几天看
- **幼猫高强度玩耍后短暂跛** —— 高活动量正常;但要排除真摔伤
- **明显单点小伤(小擦伤无肿胀)能正常走** —— 在家清洁伤口(温水/盐水)观察

---

## 六、主人能做的(限制 + 观察 + 急救)

### 急性跛行 / 急症路径
- **限制活动 —— 关在一个小房间或大笼子**,不让它跳上跳下
- **不要尝试在家正位 / 拉直腿**(可能加重损伤)
- **不要给人药止痛**(对乙酰氨基酚 / 布洛芬 / 阿司匹林对猫剧毒)
- **怀疑骨折:** 用毛巾包住固定患肢,装航空箱送医
- **开放性伤口:** 清洁纱布按压,送医
- **怀疑 ATE(后腿瘫 + 大叫 + 冰凉):** **立刻**送医,几小时窗口

### 慢性跛行(老年 OA)
- **体重管理** —— Merck 列为关键干预
- **限制跳跃高度** —— 给猫准备阶梯 / 矮处水食盆
- **温暖软的休息环境**
- **听兽医的:Frunevetmab(美国已批准的猫 OA 单抗注射)、关节保健、必要时止痛药** —— 全部交给兽医
- **物理治疗 / 关节液修饰剂** 视兽医建议

### 主人**不能**做
- **绝不给猫吃任何人用止痛药** —— 对乙酰氨基酚一片可致命(v0.2 §3.2)
- **不要在家正位 / 拉直** 怀疑骨折 / 脱位的腿
- **不要忽视老猫「不爱跳」** —— 这不是「老了」,是关节疼
- **不要让猫继续跳高 / 高强度活动** 即使它「看着没事」 —— OA 损伤是累积的

## 七、对接产品的具体改造

### 加新症状 `limp`(完整轨道)

1. **`SYMPTOM_LABELS`** 加 `limp: "跛行 / 走路异常"`
2. **`symptoms/page.tsx`** 加卡片(tier: `"common"`)
3. **`triage.ts`** 加 `limpFlow` 3 题:
   - Q1 主要表现:
     - 偶尔瘸一下,过一会就好(w1)
     - 持续一瘸一拐,某条腿不太敢用(w2)
     - 完全不能用某条腿、提着走(w2)
     - **突然后腿瘫软 + 大叫 + 后腿冰凉(ATE)redFlag**
     - 跳不上以前能跳的地方、犹豫上下(老年 OA 模式,w2)
   - Q2 多选:
     - **腿部肿胀、有伤口、明显畸形 redFlag**
     - **刚摔过 / 车祸 / 打架过 redFlag**
     - 一碰那条腿就嚎叫攻击(w2)
     - **也在发热 / 不吃 / 萎靡 redFlag**(系统感染)
     - 肉垫看到刺 / 异物(w2)
     - 脚指甲断 / 流血 / 嵌入(w2)
     - 老年猫(>10)躲低处不爱上下跳(w2)
     - 幼猫疯玩后偶尔瘸(w1)
     - 都没有(w0)
   - Q3 时长(今天 / 几天到一周 / 超一周或反复)
4. **`report/page.tsx`** 加 `limp` 组三档:
   - **red:** ATE / 骨折脱位 / 外伤后 / 系统感染征象 —— 立刻就医,**不要在家正位**,**不要喂止痛药(对乙酰氨基酚剧毒)**
   - **yellow:** 持续跛行 / 老年 OA 模式 / 肉垫异物 —— 约门诊 + 兽医 X 光 + 体重管理 + 限制跳跃 + 反对人止痛药
   - **green:** 偶发瘸一下立刻好 / 幼猫高活动 —— 限制活动几天 + 观察
5. **`NO_GREEN_GROUPS` 不加 limp** —— 偶发轻微跛行可在家观察是合理的

### 文案要点

- **「猫的脾气」与「老了不爱动」的两个隐性误区** —— 都是疼痛的隐性表达
- **「ATE 几小时窗口」** —— 后腿瘫 + 冰凉 + 大叫的组合要点出,**这种 case 拖不起**
- **「对乙酰氨基酚一片可致命」** —— 重复强化(已在 ingest 提过,limp red 再点一次)
- **「不要在家正位」** —— 骨折 / 脱位的明确禁忌
- **「60-90% 老猫有 OA」** —— 让用户重新看待「老了不爱跳」的现象
- **「Frunevetmab 单抗」** 不点名(国内可及性未知,交给兽医)

## 八、待执业兽医核对

1. **ATE 在国内宠物医院的常见诊断率 + 治疗方案 / 价格** —— 是否要在 red 文案里给「就诊预期」(可能需要超声查 HCM、抗血小板药物、严重病例预后差需准备情绪)。
2. **Frunevetmab(舒利亚)** 在国内的批准 / 可及性 —— 是否值得在黄档点名「这是新的猫 OA 单抗,可问兽医」。
3. **「猫用关节保健粉」** 商品在国内宠物圈的盛行 —— 是否要点名某些值得 / 不值得。Merck 提到关节液修饰剂(joint fluid modifiers)但效果证据等级。
4. **「老年猫跳跃减少」** 作为 OA 筛查的灵敏度 + 老年体检里建议每年做 X 光评估的兽医实践。
5. **国内宠物医院 X 光 / CT 价格区间** —— 用户对成本可能恐惧而拖延。
